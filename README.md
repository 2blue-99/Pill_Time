# 💊Pill Time (개발중)
<br/>
<br/>  

## 프로젝트 소개  

AI 객체 탐지 기능을 적용시켜 정기적인 약 및 영양제 복용을 돕는 알람 애플리케이션

## 개발정보
- 사용언어 : Kotlin
- 아키텍쳐 및 라이브러리
  - Jetpack Compose
  - Alarm Manager
  - CA
  - MVVM
  - DI
  - Flow
  - Coroutine
  - Room
  - Retrofit

<br/>
<br/>  
<br/>  

## 기능정의
- 객체 탐지 모델을 이용하여 이미지 알약 식별 및 개수 확인을 통한 알람 종료 기능  
- 진동, 소리를 통한 알람 기능  
- 알람 종료를 위한카메라를 통한 알약 객체 탐지 기능  
- 알람 추가 및 수정, 삭제, On / Off 기능
